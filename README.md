# Ideological-and-Political-Education-Case-Generation-and-Evaluation-Dataset
## datasets中为本文所有数据集
### IPEC-GE
该数据集包含839条覆盖《数据结构》课程知识点的结构化思政案例，通过“大语言模型初筛—领域专家精细化校准”的协同流水线构建。具体而言，首先利用大规模语言模型生成覆盖广泛知识点的候选案例，随后基于认知流畅性理论框架，由马克思主义理论与计算机科学交叉领域的专家团队，从逻辑自洽性与语义衔接性两个维度，对每条案例进行双盲评审与量化评分，并依据统一标准划分为高、中、低三个质量等级。此过程建立了严格的“生成-审核-反馈”质量闭环，有效解决了通用大模型在专业领域可能出现的内容失真与价值失序风险，确保了案例的专业性与思想性。

IPEC-GE数据集填补了课程思政案例生成领域高质量、细粒度标注基准数据的空白，为模型的训练与可靠评估提供了关键支撑。数据集已在GitHub平台开源，以促进研究的可复现与领域发展。 
内容具体包括初始知识点数据集和思政要素数据集以及与生成案例对应的三个数据集训练集train_new.json、测试集test_dataset.json、验证集val_dataset.json
